# 20. Valid Parentheses

## 题目

给定一个只包含括号的字符串 `s`，判断括号是否有效。

有效括号需要满足：

- 左括号必须用同类型的右括号闭合。
- 左括号必须以正确顺序闭合。
- 每个右括号都必须有一个对应的左括号。

## 思路

这题是栈最经典的基础题。

括号匹配的本质是：

```text
最近打开的左括号，必须最先被关闭
```

这正好符合栈的后进先出。

遍历字符串：

1. 遇到左括号，放进栈。
2. 遇到右括号，检查它能不能和栈顶左括号匹配。
3. 如果栈为空却遇到右括号，说明没有左括号可以匹配，返回 `False`。
4. 遍历结束后，如果栈为空，说明所有左括号都被匹配了。

## 用字典减少代码量

可以用字典保存右括号和左括号的对应关系：

```python
pairs = {
    "]": "[",
    "}": "{",
    ")": "(",
}
```

这样遇到右括号时，只需要比较：

```python
stack.pop() != pairs[char]
```

不用写很多 `if/elif` 来分别判断三种括号。

## 代码

```python
class Solution:
    def isValid(self, s: str) -> bool:
        stack: list[str] = []
        pairs = {
            "]": "[",
            "}": "{",
            ")": "(",
        }

        for char in s:
            if char in "([{":
                stack.append(char)
            else:
                if not stack:
                    return False

                if stack.pop() != pairs[char]:
                    return False

        return not stack
```

## 为什么最后要 `return not stack`

如果遍历结束后栈里还有左括号，说明它们没有被关闭。

例如：

```text
"("
"(()"
```

这些都不是有效括号。

所以最后要判断栈是否为空：

```python
return not stack
```

## 复杂度

- 时间复杂度：O(n)
- 空间复杂度：O(n)

最坏情况下，所有字符都是左括号，需要全部入栈。

## 心得

1. 括号匹配可以用栈，因为最近打开的左括号必须最先关闭。
2. 遇到左括号就入栈，遇到右括号就和栈顶比较。
3. 如果右括号出现时栈为空，说明没有左括号可以匹配。
4. 可以用字典保存括号匹配关系，减少代码量。
5. 最后要 `return not stack`，确认没有剩余未匹配的左括号。
