# 901. Online Stock Span

## 题目

设计一个 `StockSpanner` 类。

每次调用 `next(price)` 时，返回当前价格 `price` 的跨度。

跨度的含义是：

```text
从今天开始，连续往前数，有多少天的价格小于等于今天价格。
```

例如今天价格是 `75`，如果前面连续几天价格都小于等于 `75`，这些天都可以计入跨度。

## 思路

这题很适合用单调栈。

栈里存：

```python
(price, span)
```

也就是：

- `price`：某一天的股票价格
- `span`：这个价格已经合并过的跨度

当新价格 `price` 到来时，如果它比栈顶价格更大或相等：

```python
price >= self.stack[-1][0]
```

说明栈顶那一段日期都可以被当前价格覆盖。

于是把栈顶弹出，并把它的跨度加到当前 `span` 里：

```python
span += self.stack.pop()[1]
```

最后把当前价格和合并后的跨度入栈：

```python
self.stack.append((price, span))
```

## 为什么栈里存 `(price, span)`

如果只存价格，遇到新价格时虽然能知道哪些价格可以弹出，但还要一个一个数天数。

而 `(price, span)` 把一段已经合并好的连续天数压缩起来。

例如 `(70, 2)` 表示：

```text
价格 70 这一项已经代表连续 2 天
```

后面如果来了一个更大的价格，比如 `75`，就可以一次性加上这 `2` 天。

这就是这题单调栈的关键。

## 栈顶变化记录

输入：

```python
[100, 80, 60, 70, 60, 75, 85]
```

| 操作 | 当前 price | 出栈过程 | 入栈后 stack | 返回 |
| --- | --- | --- | --- | --- |
| 初始化 | - | - | `[]` | `null` |
| `next(100)` | `100` | 无 | `[(100, 1)]` | `1` |
| `next(80)` | `80` | 无，`80 < 100` | `[(100, 1), (80, 1)]` | `1` |
| `next(60)` | `60` | 无，`60 < 80` | `[(100, 1), (80, 1), (60, 1)]` | `1` |
| `next(70)` | `70` | `pop (60, 1)` | `[(100, 1), (80, 1), (70, 2)]` | `2` |
| `next(60)` | `60` | 无，`60 < 70` | `[(100, 1), (80, 1), (70, 2), (60, 1)]` | `1` |
| `next(75)` | `75` | `pop (60, 1)`, `pop (70, 2)` | `[(100, 1), (80, 1), (75, 4)]` | `4` |
| `next(85)` | `85` | `pop (75, 4)`, `pop (80, 1)` | `[(100, 1), (85, 6)]` | `6` |

## 代码

```python
class StockSpanner:
    def __init__(self) -> None:
        self.stack: list[tuple[int, int]] = []

    def next(self, price: int) -> int:
        span = 1

        while self.stack and price >= self.stack[-1][0]:
            span += self.stack.pop()[1]

        self.stack.append((price, span))
        return span
```

## 复杂度

- 单次调用最坏时间复杂度：O(n)
- 均摊时间复杂度：O(1)
- 空间复杂度：O(n)

虽然某一次 `next` 可能会弹出很多元素，但每个价格最多入栈一次、出栈一次。

所以从整体看，很多次调用的总时间是 O(n)，平均到每次就是 O(1)。

## 心得

1. 这种“当前元素可以合并前面一段连续元素”的题，很适合用单调栈。
2. 栈里不一定只存值或下标，也可以存元组。
3. 这题栈里存 `(price, span)`，把已经合并好的跨度压缩起来。
4. 当新价格大于等于栈顶价格时，就弹出栈顶，并累加它的跨度。
5. 单调栈的重点是：被弹出的元素以后不会再单独用到了，所以可以放心合并。
