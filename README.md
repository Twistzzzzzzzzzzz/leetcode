## 文件命名

每道题建议整理成两份文件：

```text
p题号_英文题目.py   # 代码
p题号_英文题目.md   # 思路和心得
```

例如：

```text
arrays-hashing/p0001_two_sum.py
arrays-hashing/p0217_contains_duplicate.md
```

## 每题记录

每道题的思路和心得建议写在同名 `.md` 笔记里，`.py` 文件主要保留可运行代码。

笔记里建议至少写清楚：

- 题目在考什么
- 自己的解题思路
- 时间复杂度
- 空间复杂度
- 容易错的地方
