# 238. Product of Array Except Self

## 题目

给定一个数组 `nums`，返回一个数组 `answer`。

其中：

```text
answer[i] = nums 中除了 nums[i] 以外所有元素的乘积
```

题目要求：

- 时间复杂度 O(n)
- 不使用除法

## 思路一：乘积 + 零数量

先算出所有非零数字的乘积，再统计数组里有几个 `0`。

分情况：

1. 如果有两个或更多 `0`，每个位置的结果都是 `0`。
2. 如果只有一个 `0`，只有这个 `0` 的位置结果是所有非零数字的乘积，其他位置都是 `0`。
3. 如果没有 `0`，每个位置可以用总乘积除以当前数字。

这个思路很好理解，但它使用了除法，不符合题目的正式限制。

可以保留它帮助理解零的情况。

## 思路二：左前缀积 + 右后缀积

这道题的正式解法是不用除法。

对于每个位置 `i`：

```text
answer[i] = 左边所有数字的乘积 * 右边所有数字的乘积
```

所以可以分两步：

1. 从左到右，先让 `result[i]` 保存 `i` 左边所有数字的乘积。
2. 从右到左，用 `right_product` 保存 `i` 右边所有数字的乘积，再乘到 `result[i]` 上。

## 代码

```python
class Solution:
    def productExceptSelf(self, nums: list[int]) -> list[int]:
        result = [1] * len(nums)

        for i in range(1, len(nums)):
            result[i] = result[i - 1] * nums[i - 1]

        right_product = 1

        for i in range(len(nums) - 1, -1, -1):
            result[i] *= right_product
            right_product *= nums[i]

        return result
```

## 例子

```text
nums = [1, 2, 3, 4]
```

第一遍从左到右，`result[i]` 记录左边乘积：

```text
result = [1, 1, 2, 6]
```

含义是：

```text
位置 0 左边没有数字，所以是 1
位置 1 左边是 1
位置 2 左边是 1 * 2
位置 3 左边是 1 * 2 * 3
```

第二遍从右到左，用 `right_product` 记录右边乘积：

```text
位置 3 右边没有数字，所以乘 1
位置 2 右边是 4
位置 1 右边是 3 * 4
位置 0 右边是 2 * 3 * 4
```

最终得到：

```text
[24, 12, 8, 6]
```

## 复杂度

- 时间复杂度：O(n)
- 空间复杂度：O(1)

如果不把输出数组算作额外空间，空间复杂度是 O(1)。

## 心得

1. 这题和前缀和很像，只是从“前缀和”变成了“前缀积”。
2. 可以用数组记录前一步的信息：`result[i]` 先保存左边所有数字的乘积。
3. 再用一个变量 `right_product` 从右往左记录右边所有数字的乘积。
4. 遇到“除了自己以外的所有元素”这类题，可以先想：能不能拆成“左边贡献”和“右边贡献”。
5. 304 更准确应该叫二维前缀和；它和这题一样，核心都是提前保存可复用的累计信息。
